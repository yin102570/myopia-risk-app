# 近视风险AI早筛系统

> 基于多模态生物标志物的智能近视风险预测平台

## 📋 项目概述

本系统是一个基于机器学习的近视风险预测Web应用，通过分析眼液中的多种生物标志物（包括缺氧、炎症、代谢、离焦等亚型指标），为儿童和青少年提供近视风险的快速评估和专业干预建议。

### 核心功能

- ✅ **实时风险评分**：输入检测数据，立即获得0-100分的风险评分
- ✅ **智能风险分级**：自动判断高风险/中风险/低风险三个等级
- ✅ **专业干预建议**：根据风险等级提供个性化的医学建议
- ✅ **示例数据演示**：内置高/中/低三种风险类型的示例数据
- ✅ **详细概率展示**：可查看高危/低危的具体概率值
- ✅ **数据重置功能**：一键清空输入，重新开始

## 🔬 算法原理

### 1. 生物标志物体系

本系统基于近视发生发展的多通路机制，构建了11个核心生物标志物：

#### 亚型占比指标（5个）
- **缺氧亚型占比 (prop_hypoxia)**：反映眼组织缺氧程度，缺氧是近视发展的重要诱因
- **炎症亚型占比 (prop_inflammation)**：反映眼部炎症反应，炎症会加速巩膜重塑
- **多巴胺亚型占比 (prop_DA)**：多巴胺是近视保护因子，含量越低风险越高
- **代谢亚型占比 (prop_metabolism)**：反映细胞代谢状态，代谢异常与近视相关
- **离焦亚型占比 (prop_defocus)**：反映视网膜离焦状态，离焦模式影响眼轴生长

#### 关键分子指标（6个）
- **HIF1α (pg/mL)**：缺氧诱导因子，缺氧环境下的核心调控分子
- **IL1β (pg/mL)**：白介素-1β，炎症反应的关键细胞因子
- **MMP2 (ng/mL)**：基质金属蛋白酶-2，参与巩膜基质降解和重塑
- **DA (ng/mL)**：多巴胺，视网膜分泌的近视抑制因子
- **Lactate (mmol/L)**：乳酸，糖酵解产物，反映代谢状态
- **RORA (pg/mL)**：视黄酸受体相关孤儿受体α，昼夜节律调控因子

### 2. 机器学习模型

使用**随机森林分类器**（Random Forest Classifier）作为核心预测模型：

**优势**：
- 处理高维非线性关系能力强
- 抗过拟合性能好
- 可解释性高（特征重要性）
- 对数据缺失和异常值有鲁棒性

**训练流程**：
```
原始数据 → 数据清洗 → 特征标准化 → 训练集/测试集划分
→ 随机森林训练 → 模型评估 → 保存模型 (risk_model.pkl)
```

**模型评估指标**：
- 准确率 (Accuracy)
- 精确率 (Precision)
- 召回率 (Recall)
- F1分数 (F1-Score)

### 3. 风险分级逻辑

```
风险评分 = model.predict_proba(输入数据) × 100

if 风险评分 ≥ 70:
    风险等级 = "高风险"
    颜色标识 = 红色 (#e74c3c)
    建议 = 增加户外光照+定期眼科检查，使用低剂量抗VEGF干预

elif 风险评分 ≥ 30:
    风险等级 = "中风险"
    颜色标识 = 橙色 (#f39c12)
    建议 = 减少近距离用眼时长，每天保证2小时户外活动

else:
    风险等级 = "低风险"
    颜色标识 = 绿色 (#2ecc71)
    建议 = 维持现有用眼习惯，每6个月复查一次
```

## 📊 数据流程

### 输入数据处理

```python
# 1. 数据采集
用户输入11个生物标志物数值

input_data = pd.DataFrame({

    'prop_inflammation': [value],
    ...

input_scaled = scaler.transform(input_data)

# 3. 模型预测
risk_prob = model.predict_proba(input_scaled)[0, 1]

# 4. 风险评分
risk_score = int(risk_prob * 100)

# 5. 结果输出
返回：风险评分、风险等级、专业建议、详细概率
```

### 示例数据设计

系统内置了三套示例数据，覆盖不同风险等级：

| 风险类型 | HIF1α | IL1β | DA | 缺氧占比 | 炎症占比 | 风险评分 |
|---------|-------|------|-----|----------|----------|----------|
| 高风险 | 45.0 | 18.0 | 2.2 | 0.38 | 0.28 | 85分+ |
| 中风险 | 30.0 | 12.0 | 3.5 | 0.25 | 0.22 | 30-70分 |
| 低风险 | 15.0 | 7.0 | 4.8 | 0.12 | 0.15 | 30分以下 |

**设计逻辑**：
- **高风险**：缺氧、炎症指标高，多巴胺低，亚型失衡明显
- **中风险**：各项指标介于高低之间，处于临界状态
- **低风险**：缺氧、炎症指标低，多巴胺正常，亚型平衡

## 📖 使用指南

### 基本操作流程

1. **输入检测数据**
   - 手动输入11个生物标志物的检测值
   - 或从侧边栏选择加载示例数据

2. **计算风险评分**
   - 点击「计算风险评分」按钮

3. **查看结果**
   - 风险评分（0-100分）
   - 风险等级（高/中/低）
   - 专业干预建议
   - 详细概率（可展开查看）

4. **重置数据**
   - 点击「重置输入数据」按钮
   - 清空所有输入，重新开始

### 参数说明

#### 生物标志物指标

| 指标 | 单位 | 正常范围 | 高风险提示 |
|------|------|----------|-----------|
| HIF1α | pg/mL | 10-50 | >40: 重度缺氧 |
| IL1β | pg/mL | 5-20 | >15: 炎症活跃 |
| MMP2 | ng/mL | 0.3-1.2 | >1.0: 巩膜重塑 |
| DA | ng/mL | 2.0-5.0 | <2.5: 多巴胺不足 |
| Lactate | mmol/L | 1.5-3.0 | >2.5: 代谢异常 |
| RORA | pg/mL | 10-30 | <15: 昼夜紊乱 |

#### 亚型占比

所有亚型占比之和应接近1.0（100%），不同亚型的相对比例反映近视发生的生理机制。

### 1. 性能优化

#### 模型缓存

@st.cache_resource(ttl=3600)
def load_model_and_scaler():
    """缓存1小时，避免重复加载"""
    model = joblib.load("risk_model.pkl")
    scaler = joblib.load("scaler.pkl")
    return model, scaler

## 📚 医学背景

### 近视发病机制

近视的发生发展是一个多因素、多通路参与的复杂过程，主要包括：

1. **缺氧通路**：巩膜缺氧导致胶原合成减少，眼轴增长
2. **炎症通路**：炎症因子激活MMPs，降解巩膜基质
3. **代谢通路**：糖酵解异常导致能量代谢紊乱
4. **神经通路**：多巴胺减少失去对眼轴的抑制
5. **光学通路**：周边离焦模式影响眼轴生长方向

### 生物标志物选择依据

本系统选择的11个生物标志物均基于权威文献和临床研究：

- **HIF1α**：近视缺氧机制的核心分子
- **IL1β**：炎症通路的标志性因子
- **MMP2**：巩膜重塑的关键酶
- **DA**：多巴胺是近视保护因子
- **Lactate**：代谢状态的敏感指标
- **RORA**：昼夜节律与近视的关联分子

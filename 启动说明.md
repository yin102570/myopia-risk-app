# AI近视风险预测系统 - 启动说明

## 快速启动步骤

### 方法1: 使用启动脚本（推荐）

1. 双击运行 `启动系统.bat` 文件
2. 等待服务器启动（约3-5秒）
3. 浏览器会自动打开 http://localhost:3000
4. 开始使用系统

### 方法2: 手动启动

1. 打开命令行终端（CMD或PowerShell）
2. 进入项目目录：
   ```bash
   cd d:\HuaweiMoveData\Users\asdf1\Desktop\TEST\myopia-risk-ai
   ```
3. 启动服务器：
   ```bash
   python standalone_server.py
   ```
4. 打开浏览器访问：http://localhost:3000

## 系统功能

### 1. 风险预测
- 输入11项眼部参数和生活方式数据
- AI模型预测近视风险等级（低/中/高）
- 获取个性化的防控建议

### 2. 仪表盘
- 查看预测统计数据
- 模型性能指标
- 风险分布图表

### 3. 数据生成
- 使用GAN生成虚拟数据
- KS检验验证数据质量
- 扩充训练数据集

### 4. 模型训练
- 多种模型选择（随机森林、逻辑回归等）
- 实时训练进度
- 模型性能评估

## 技术栈

- **后端**: Flask + Python + Scikit-learn
- **前端**: React 18 + Material-UI + Recharts
- **AI模型**: 随机森林分类器
- **数据生成**: GAN（生成式对抗网络）

## 停止服务器

在运行服务器的终端窗口中按 `Ctrl+C` 即可停止服务器。

## 常见问题

### Q: 服务器启动失败？
A: 确保已安装Python 3.7+和必要的依赖包：
```bash
pip install flask flask-cors numpy pandas scikit-learn
```

### Q: 端口3000被占用？
A: 可以修改 `standalone_server.py` 最后一行的端口号：
```python
app.run(host='0.0.0.0', port=8080, debug=False)  # 改为8080或其他端口
```

### Q: 首次访问速度慢？
A: 首次运行需要训练模型，请等待约5-10秒。模型会自动保存，后续启动会更快。
